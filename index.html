CERITA & ILUSTRASI PANEL DENGAN KONSISTENSI KARAKTER — DITENAGAI GEMINI API

🔑 API Key Gemini

Dapatkan API key gratis di aistudio.google.com/apikey. Key disimpan hanya di memori browser kamu selama sesi ini — tidak dikirim atau disimpan ke server manapun selain langsung ke Google. Kamu bisa pakai key yang sama untuk keduanya, atau key/akun berbeda supaya kuota cerita dan gambar tidak saling mengurangi.

Tier gratis Gemini API punya kuota harian & rate limit per menit terpisah untuk tiap API key/project. Kalau kena limit di salah satu, yang lain tetap jalan kalau pakai key berbeda.

📦 Proyek

Simpan progres sebagai file JSON di komputer kamu, atau lanjutkan proyek yang sudah ada. Aplikasi ini tidak menyimpan apa pun otomatis — export manual sebelum menutup tab.

✏️ Buat Cerita

Jelaskan ide cerita kamu. Gemini akan menyusun premis lengkap, daftar karakter, dan breakdown chapter + panel yang siap dipakai untuk generate gambar.

🎭 Character Bible

Generate gambar referensi "master" untuk tiap karakter dulu — gambar ini akan dipakai ulang sebagai acuan supaya wajah, outfit, dan ciri khas karakter tetap konsisten di semua panel.

🖼️ Generate Gambar Panel

Tiap panel akan digenerate memakai gambar referensi karakter yang terlibat, supaya konsisten dengan Character Bible. Generate satu-satu atau sekaligus per chapter.

📱 Preview Webtoon

Tampilan scroll vertikal seperti webtoon. Klik tombol di bawah untuk download tiap gambar panel satu per satu.
